## 📱 Overview

হিসাবঘর Pro is a cutting-edge Android business management application designed specifically for Bangladesh shop owners. Built with Flutter and Firebase, it offers a premium bilingual experience (Bangla + English) with seamless language switching, offline-first architecture, and cloud backup capabilities.

Perfect for:
- 🏪 Grocery Shops
- 💊 Pharmacy
- 👔 Clothing Stores
- 🍽️ Restaurants
- 🏬 Small Businesses
- 📦 Wholesale Shops

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (Latest Stable Version)
- Android Studio
- Firebase Project
- Google Cloud Console Setup
- SMS API Credentials (SSL Wireless / BulkSMSBD)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd hisabghor-pro

# Install dependencies
flutter pub get

# Configure Firebase
# Follow instructions in README_SETUP.md

# Run the app
flutter run
```

### Build APK

```bash
# Debug APK
flutter build apk --debug

# Release APK
flutter build apk --release

# App Bundle (Play Store)
flutter build appbundle --release
```

📖 **Detailed setup instructions**: See `README_SETUP.md`
